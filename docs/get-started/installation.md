# Installation

To install *Kornia*, you can do it in two different ways: using the provided [PyPi](https://pypi.org/project/kornia/) wheels or directly from source.

!!! note
    *Kornia only has as a dependency Pytorch.*

1. From pip:

```bash
pip install kornia
```

2. From source:

```bash
python setup.py install
```

3. From source using pip:

```bash
pip install git+https://github.com/kornia/kornia
```

Once you succeeded installing *Kornia* check you can import:

```bash
python -c "import kornia; print(kornia.__version__)"
```
